Currency Converter

Описание:
Простой и удобный конвертер валют, который позволяет быстро перевести любую сумму из одной валюты в другую с использованием актуальных курсов.

Функционал:
Выбор исходной и целевой валюты (USD, EUR, RUB, KGS, INR).
Ввод суммы для конвертации.
Кнопка Convert для расчёта и отображения результата.
Кнопка ⇄ для мгновенной смены валют местами.
Автоматическое обновление результата при изменении валют или суммы.
Валидация ввода: проверка на корректность введённого числа.
Показ актуального курса: 1 USD = 0.92 EUR.
Сообщения об ошибках при некорректном вводе или сбое API.

Технологии:

HTML / CSS для структуры и стиля.
JavaScript для работы с API и динамического отображения результата.
ExchangeRate-API для получения актуальных курсов валют.

Особенности дизайна:

Тёмная современная тема с акцентом на результат.
Адаптивная верстка для мобильных и десктопных устройств.
Плавные анимации при отображении результата.

Использование:

Выберите валюту From и To.
Введите сумму для конвертации.
Нажмите Convert или клавишу Enter.
При необходимости поменяйте валюты местами кнопкой ⇄.
Результат и курс отображаются ниже ввода.
